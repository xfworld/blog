# 进阶分析参考：行业专项问题框架

## 科技 / 新能源 / 汽车行业专项问题

### 收入结构转型分析

当公司处于"从卖硬件到卖订阅/软件/服务"的转型期时，重点追问：

1. **软件渗透率（Take Rate）**
   - 当前渗透率是多少？（用户基数 vs 订阅用户数）
   - 若渗透率翻倍，软件业务对净利润的弹性有多大？（因软件边际成本≈0，增量收入几乎全部转化为毛利）
   - 类比参考：微软 Office 365 订阅化后毛利率从 62% 升至 69%；Adobe CC 转型后 P/E 从 20x 升至 50x+

2. **新平台 / 新产品的单位经济模型**
   - 单车/单用户的 CAC（获客成本）vs LTV（生命周期价值）
   - 下一代平台的 BOM 成本降低目标是否有供应链协议支撑？
   - 产能爬坡曲线：参考第一性原理，达到目标产能的时间节点

3. **新产品线毛利转正时间表**
   - 产能地狱期：产量低时固定成本摊薄不足，毛利率为负
   - 转正信号：关注单季度产量是否突破盈亏平衡点
   - 历史参考：Model 3 于 2018Q3 实现毛利转正（交付量突破 5 万辆/季）

## AI / 算力投资 ROI 评估框架

当公司大量 Capex 投入 AI 基础设施时：

### 两种商业化路径对比

| 路径 | 商业模式 | 估值方法 | 变现时间 |
|------|---------|---------|---------|
| **内部降本**：AI 替代人工 | 成本节约 → 利润率提升 | 按节约成本折现 | 1-3 年 |
| **对外卖服务**：算力云 / API | SaaS/云 收入 | P/S 20-50x | 2-5 年 |

### ROI 计算框架
```
年化 ROI = (AI带来的年化收益 / 累计AI投入) × 100%

收益可来自：
- FSD/自动驾驶授权收入
- 机器人替代人工节省的劳动力成本  
- 算力对外销售收入
- 数据变现（训练数据授权）
```

## 机器人 / 自动化业务估值方法

当公司机器人业务从实验室进入规模化生产时：

### 估值锚点
- **制造业替代**：全球制造业人工成本约 $17 万亿/年，机器人即使替代 1% 也是 $1700 亿市场
- **单台经济模型**：COGS / ASP / Payback Period（投资回收期）
- **对标案例**：Boston Dynamics、ABB、发那科的 EV/EBITDA 区间（12-25x）

## 地缘政治风险量化框架

### 单一市场 / 工厂依赖度评估
```
暴露度 = 该工厂营收占比 × 受影响概率 × 替代迁移成本系数
```

典型情景：
- 对华加征关税 25%：若上海厂占总产能 X%，则影响毛利 = X% × 产量 × 25% × 单车价格
- 供应链重构成本：建设同等产能新工厂通常需要 2-4 年、$30-50 亿资本开支

## Robotaxi / 出行服务估值重构

### 从"卖车"到"卖里程"的估值范式切换

| 维度 | 汽车制造商 | 出行平台 |
|------|-----------|---------|
| 估值方法 | P/E（8-15x） | EV/Gross Profit（30-80x）|
| 核心指标 | 交付量、毛利率 | GMV、Take Rate、Rides/Day |
| 资产属性 | 重资产（卖出去） | 轻资产（服务化） |

### Robotaxi 估值模型要素
1. **车队规模**：N 辆车 × 每日里程 × 每英里费率
2. **Take Rate**：平台从每笔收入中抽成比例（Uber ≈ 28%）
3. **可用率**：无人驾驶车辆可 24/7 运营，利用率远高于私家车（私家车≈5%，Robotaxi 可达40-60%）
4. **折旧**：车辆资产留在公司表内，需扣除折旧

**用户视角对比**（买车 vs 订阅出行）：
- 买车总成本（TCO）= 购车价 + 保险 + 维修 + 停车 + 折旧 ≈ $1.2-1.8 万/年（美国均值）
- 若 Robotaxi 定价低于 TCO，将触发大规模私家车替代
- 临界价格点 ≈ $0.40-0.60/英里（现 Uber/Lyft 约 $1.5-2.5/英里）

## FSD 软件净利润弹性测算

### 情景假设模板

```
基础数据：
- 在路车辆数（Vehicle Parc）：N 辆
- FSD 当前渗透率：P₀%
- FSD 月订阅价：$X 或 一次性 $Y

情景测算（渗透率从 P₀ 升至 P₁）：
- 新增订阅用户 = N × (P₁ - P₀)
- 新增年化收入 = 新增用户 × $X × 12（或 × $Y）
- 软件毛利率 ≈ 90%+（边际成本极低）
- 新增毛利 ≈ 新增收入 × 90%
- 税后影响（税率25%）= 新增毛利 × 75%
- EPS 提升 = 新增税后净利 / 摊薄股数
```

## 能源业务 vs FSD 软件业务对比模板

| 维度 | 能源业务（储能+光伏） | FSD 软件业务 |
|------|---------------------|------------|
| 商业模式 | 硬件销售 + 电网服务 | 订阅 / 一次性授权 |
| 毛利率 | 20-30%（硬件拖累） | 80-95% |
| 增长驱动 | 电网投资周期、政策补贴 | 车辆保有量 × 渗透率 |
| 估值倍数 | P/E 15-25x（工业属性） | P/S 10-30x（SaaS属性）|
| 对汽车依赖 | 低（独立供能市场） | 高（依赖车辆销量积累基数）|
| 现金流特征 | 项目制，现金流波动大 | 订阅制，现金流极稳定 |

**结论逻辑**：两者共同作用是降低"卖车数量"对营收的决定性权重，使公司在汽车销量承压时仍能保持利润增长，从而支撑更高的估值倍数。

## 股价泡沫诊断框架

### 四维检验法

1. **绝对估值法**：当前 P/FCF vs 历史区间。若当前值处于历史 90th percentile 以上，需合理解释超额溢价来源。

2. **反推增长法（Reverse DCF）**：用当前股价反推市场隐含的未来增速假设，再问这个增速现实吗？
   - 公式：隐含增速 = [(市值/FCF) × WACC - 1]^(1/n) - 1（简化）

3. **叙事溢价拆解**：
   - 当前业务合理估值（基于 DCF）
   - 期权价值（FSD、Robotaxi、Optimus 等）
   - 若"期权价值"占总市值 >50%，说明股价高度依赖叙事兑现

4. **市场情绪指标**：
   - 散户持仓集中度
   - 期权市场隐含波动率（IV）
   - 做空比例变化趋势
