## 核心问题

每次分析至少回答这些问题：

1. 本年度最核心的收入来源是什么？收入是否依赖单一产品、单一客户、单一地区、补贴或周期价格？
2. 与去年相比，毛利率发生了什么变化？拆成价格、销量、成本、产品结构、汇率、产能利用率和会计口径变化。
3. 财报披露的主要经营风险有哪些？哪些是模板化风险，哪些已经在数字里出现？
4. 经营活动现金流是否覆盖净利润？若不能，解释差额来自应收账款、存货、预付款、递延收入、股权激励、税费还是一次性项目。
5. 资本开支是否在创造未来现金流，还是在维持旧业务？用 `FCF = OCF - Capex` 检查自由现金流。
6. 资产负债表有没有异常膨胀的科目？重点看应收、存货、其他应收款、预付款、商誉、长期股权投资、递延所得税资产、货币资金受限。
7. 管理层说的增长故事是否能被订单、交付、客户留存、订阅率、产能爬坡、价格、现金流和同行数据验证？
8. 当前估值隐含了什么增长率、利润率和成功概率？市场是在买现实，还是在买远期叙事？

## 小白解释规则

- 第一次出现专业词时，用一句话解释。例：`应收账款`就是货卖出去了但钱还没收回来，利润表先记了收入，钱包却还没变厚。
- 解释逻辑链，不只解释名词。例：应收账款增速高于收入增速 -> 公司卖货更依赖赊账 -> 现金回款变慢 -> 坏账风险上升 -> 利润质量下降。
- 用生活类比，但不要替代数据。例：毛利率像饭店每卖一桌菜扣掉食材后的剩余比例，能反映定价权和成本控制。
- 给出可复用方法。例：不是只说“库存高危险”，要说“看库存增速是否超过收入增速、库存周转天数是否变长、是否计提跌价准备”。

## 财报陷阱清单

### 1. 纸面富贵：利润好，但钱回不来

- 识别：应收账款增速远超营业收入增速，经营活动现金流长期低于净利润。
- 真相：可能提前确认收入、放宽信用、向渠道压货，利润先出现，现金未到账。
- 追问：账龄是否恶化？坏账准备是否充分？前五大客户是否集中？期后回款如何？

### 2. 虚资产：存货和应收账款爆棚

- 识别：存货周转率下降、存货跌价准备不足、长账龄应收占比上升。
- 真相：存货可能过时，欠款可能收不回，但账面仍按较高价值显示。
- 追问：库存结构是什么？原材料、在产品、产成品谁在增加？是否存在渠道退货或价格下跌？

### 3. 财务黑盒：其他应收款与预付款项

- 识别：其他应收款占总资产比例异常，预付款与采购规模不匹配。
- 真相：可能是关联方资金占用、变相借款、提前付款转移资金。
- 追问：交易对手是谁？是否关联方？是否有商业必要性？是否长期挂账？

### 4. 商誉雷区：高价收购后的幻象

- 识别：商誉/净资产过高，例如超过 30% 需重点警惕。
- 真相：被收购资产业绩下滑时，商誉减值可能一次性吞掉多年利润。
- 追问：业绩承诺完成了吗？是否踩线完成？被收购业务现金流是否真实？

### 5. 大存大贷：现金很多却高息借款很多

- 识别：货币资金与短期借款同时高企，利息收入低于合理水平，受限资金比例高。
- 真相：账上钱可能不可动用、被质押、被冻结，极端情况下现金真实性有问题。
- 案例边界：康美药业说明现金造假风险真实存在，但不能把所有大存大贷公司都直接判为造假；要结合受限资金、利息收入、审计意见和银行函证。

## 隐蔽陷阱清单

### 1. 研发支出资本化：把亏损藏进资产

- 逻辑：费用化会立刻减少利润，资本化会进入资产并以后摊销。
- 识别：研发资本化比例显著高于同行，开发支出快速增长，后续商业化证据不足。
- 解释：这像把本月房租说成“未来资产”，短期利润好看，但未来可能要摊销或减值。

### 2. 折旧政策微调：不流血的利润提升

- 逻辑：延长固定资产使用年限会降低当期折旧费用，从而抬高利润。
- 识别：会计估计变更、折旧年限调整、残值率变化、单位产能折旧下降。
- 追问：设备真实寿命是否变长？同行是否类似？产能利用率是否支撑这个判断？

### 3. 联营企业隐藏亏损：表外黑洞

- 逻辑：未并表公司只体现长期股权投资和投资收益，不完整呈现其债务。
- 识别：长期股权投资巨大，投资收益长期为负或很低，关联担保或承诺很多。
- 追问：联营公司承担了哪些烧钱业务？上市公司是否有回购、担保、继续注资义务？

### 4. 减值拨备洗大澡

- 逻辑：亏损年份一次性计提大量减值，把未来包袱提前清掉。
- 识别：巨额商誉、存货、坏账、固定资产减值与管理层换届、战略调整同时出现。
- 双面解读：当年很差，但若减值充分，未来利润基数可能变低；要判断是风险暴露还是财务重置。

## 投资判断框架

1. 先判商业模式：收入是否可持续、是否有定价权、是否有规模经济、是否需要持续烧钱。
2. 再判财务质量：利润是否变成现金，资产是否干净，负债是否可承受。
3. 再判增长质量：增长来自真实需求、价格上涨、份额提升、并购、补贴还是会计处理。
4. 再判估值：把成熟业务按合理倍数估值，把高不确定业务按情景和概率估值。
5. 最后给投资参考：明确适合哪类投资者、主要跟踪指标、买入需要的安全边际、卖出或回避信号。

常用估值方法：

- 成熟盈利公司：P/E、EV/EBIT、DCF。
- 重资产周期公司：P/B、EV/EBITDA、周期中枢利润。
- 高增长软件或订阅业务：ARR、净收入留存、Rule of 40、EV/Sales 与长期净利率假设。
- 金融公司：P/B、ROE、资本充足率、信用成本。
- 亏损但有期权价值公司：分部估值、概率加权情景估值、实物期权思路。

## 特斯拉式深挖问题模板

这些问题适合分析存在“硬件业务 + 软件/AI/能源/平台期权”的公司，不只适用于特斯拉。

### 收入结构：从卖硬件到卖订阅

- 核心问题：汽车或硬件收入占比是否仍过高？软件、订阅、服务、能源或平台收入是否足以改变估值倍数？
- 订阅转化率：FSD 或类似软件的 take rate 当前是多少？拐点来自价格下降、功能提升、监管放行、保险折扣还是车队规模？
- 利润弹性：如果订阅率达到 50%，新增收入中多少会变成毛利？需要扣除算力、客服、责任保险、监管合规和持续研发成本，不能简单假设软件零成本。

### 盈利质量：成本控制与技术溢价

- 定价权是否被竞争者削弱？降价带来的销量增量是否补得回毛利率损失？
- 下一代低成本平台的单车成本目标是多少？成本下降来自电池、制造工艺、零部件数量、工厂自动化、供应链还是规模采购？
- 新产品何时毛利转正？产能爬坡期亏损是否拖累整体毛利率，转正时间由产量、良率、采购成本和定价决定。

### 研发与 AI：资产负债表外的隐形价值

- AI/算力/芯片/机器人投入的 ROI 如何衡量？是提升主业务效率、卖软件订阅、卖云服务，还是形成新硬件品类？
- Dojo 或自研算力是否比外部 GPU 更便宜、更高效？用训练成本、迭代速度、模型能力、FSD 订阅收入和安全指标验证。
- 机器人何时从实验室进入生产线？先看内部部署数量、可执行任务复杂度、替代人工成本、故障率和维护成本，再谈外部销售。

### 风险与监管

- 地缘政治、关税、补贴变化如何影响供应链、产能布局和单车毛利？
- 自动驾驶或 Robotaxi 是否有明确法律准入路径？没有监管准入，技术领先不能直接变现。
- 数据安全、召回、事故责任、保险成本和地方牌照是否会改变商业模式。

### Robotaxi 或租赁化估值

- 如果自动驾驶就绪，公司可能从卖车转向运营车队、租赁、按里程收费或平台抽佣。
- 估值应从一次性卖车利润转向车辆生命周期现金流：单车日均运营小时、每英里收入、利用率、维护成本、保险、清洁、折旧、电费、平台抽成、监管成本。
- 对用户的价值主张：免去购车首付、车位费、保险、折旧和软件订阅，换成按需出行费用。关键是总拥有成本 TCO 是否明显低于买车或网约车。
- 风险：资产更重、监管更强、事故责任更集中，不能直接套用轻资产软件倍数。

### 能源业务 vs FSD 软件模式

- 能源业务：通常更像项目或设备业务，受订单、产能、原材料、安装周期和电网需求影响；优点是市场大、与新能源生态协同，缺点是资本开支和交付周期更重。
- FSD 软件：更像高毛利订阅业务，边际成本低，估值倍数可能更高；前提是技术可用、监管准入、责任成本可控、用户愿意持续付费。
- 降低卖车依赖的路径不同：能源靠第二增长曲线和现金流多元化，FSD 靠提高单车生命周期利润和估值倍数。

## 输出口径

- 结论必须分层：事实、计算、推断、投资判断。
- 每个关键判断至少给一个反证条件。例：如果 FSD 订阅率连续两个季度没有提升，则软件估值重估逻辑需要下修。
- 不要只列风险，要说明风险如何进入利润表、现金流量表或估值模型。
- 对普通投资者的建议要可执行：跟踪哪些指标、下一份财报看什么、什么情况下观望或回避。
