## 德说-第451期, 阿里突然成立“悟空事业部”：真正的 AI 战争已转向B端工作系统  
                
### 作者                
digoal                
                
### 日期                
2026-03-16                
                
### 标签                
AI , 操作系统 , 工作流                
                
----                
                
## 背景                
今天一条看似简单的新闻，其实可能是中国 AI 产业最重要的一次战略信号。  
  
根据《科创板日报》消息：  
  
* Alibaba 成立 **Alibaba Token Hub（ATH）事业群**  
* 由 CEO **吴泳铭** 直接负责  
* 内部首次曝光 **“悟空事业部”**  
* 目标：打造 **B端 AI 原生工作平台**  
  
同时：  
  
* C端：已经有 **通义千问** APP  
* B端：开始做 **AI 原生工作流系统**  
  
很多人看到这里会觉得：  
  
> “这不就是企业版AI助手吗？”  
  
如果你这么理解，就完全低估了这件事。  
  
**这不是一个产品升级，而是一场“操作系统级别”的战争。**  
  
今天我们从第一性原理出发，讲清楚三件事：  
  
1. 为什么 **AI竞争一定会从C端转向B端**  
2. 阿里为什么突然押注 **AI工作流平台**  
3. 未来10年 **最大的AI公司不会是模型公司**  
  
   
  
## 一、第一性原理：AI的真正价值在哪里？  
  
先讲一个最底层的经济学规律。  
  
**技术的价值 = 是否进入生产系统**  
  
举几个历史例子：  
  
| 技术    | 真正爆发点   |  
| ----- | ------- |  
| 电力    | 工厂流水线   |  
| 计算机   | 企业ERP系统 |  
| 互联网   | 电商与广告   |  
| 移动互联网 | 支付与社交   |  
  
规律非常清晰：  
  
**技术只有进入生产系统，才会产生巨大价值。**  
  
所以 AI 也是一样。  
  
AI如果只是：  
  
* 写文章  
* 生成图片  
* 聊天  
  
它只是一个 **消费级娱乐产品**。  
  
但如果 AI 进入：  
  
* 企业运营  
* 供应链  
* 财务  
* 研发  
* 销售  
  
那就变成 **生产力革命**。  
  
这就是阿里这次调整的核心逻辑。  
  
   
  
## 二、为什么阿里一定要做 AI 工作平台？  
  
因为 AI 的最大市场 **不是 C端，而是 B端**。  
  
看一个非常重要的数据。  
  
全球企业软件市场规模：  
  
* **企业软件（SaaS）** ：约 **9000亿美元**  
* **AI消费应用**：不到 **1000亿美元**  
  
而且利润率更高。  
  
典型例子：  
  
| 公司         | 估值      |  
| ---------- | ------- |  
| Salesforce | 2000亿美元 |  
| SAP        | 1800亿美元 |  
| ServiceNow | 1500亿美元 |  
  
但这些公司有一个共同问题：  
  
**系统非常笨。**  
  
原因很简单：  
  
传统软件逻辑是：  
  
```  
人 -> 软件 -> 数据 -> 结果  
```  
  
AI时代逻辑变成：  
  
```  
人 -> AI -> 自动完成工作  
```  
  
这就是 **AI Native Software（AI原生软件）** 。  
  
所以阿里这次成立 **ATH + 悟空事业部**，本质是要做：  
  
> **AI时代的企业操作系统**  
  
   
  
## 三、真正的 AI 机会：不是模型，而是“工作流”  
  
很多人误判了 AI 竞争。  
  
大家都在讨论：  
  
* 谁的模型更强  
* 谁参数更多  
* 谁算力更大  
  
但真正的商业价值，往往在 **应用层**。  
  
历史已经证明这一点。  
  
当年互联网时代：  
  
最赚钱的不是基础技术公司，而是平台公司：  
  
| 技术层  | 公司        |  
| ---- | --------- |  
| 协议   | TCP/IP    |  
| 操作系统 | Microsoft |  
| 平台   | Amazon    |  
| 应用   | Google    |  
  
AI时代也是一样。  
  
技术栈大致是：  
  
```  
算力层  
↓  
模型层  
↓  
Agent层  
↓  
工作流层  
↓  
行业应用  
```  
  
**最有价值的是“工作流层”。**  
  
原因很简单：  
  
企业真正的需求不是：  
  
> “有个AI能聊天”  
  
而是：  
  
> “AI帮我把事情做完”  
  
例如：  
  
销售流程：  
  
```  
客户线索  
↓  
AI分析  
↓  
生成方案  
↓  
报价  
↓  
合同  
↓  
交付  
```  
  
如果 AI 能自动完成 80%，企业效率会提升 **3-10倍**。  
  
这就是 AI 的真正价值。  
  
   
  
## 四、为什么阿里最有可能做成这件事？  
  
因为它拥有 **三个关键资源**。  
  
### 第一：企业数据  
  
阿里有中国最大的企业数据体系：  
  
* 阿里云  
* 钉钉  
* 电商供应链  
  
钉钉企业用户：  
  
**超过2300万组织**  
  
这意味着：  
  
**中国最多的企业工作数据在阿里。**  
  
   
  
### 第二：工作入口  
  
企业软件最重要的是 **入口**。  
  
例如：  
  
* Slack  
* Teams  
* 飞书  
  
阿里的入口就是：  
  
**钉钉**  
  
如果 AI Agent 直接在钉钉里运行：  
  
```  
聊天  
↓  
触发AI  
↓  
自动执行任务  
↓  
生成结果  
```  
  
那就是 **AI操作系统**。  
  
   
  
### 第三：模型能力  
  
阿里拥有：  
  
* 通义千问  
* 大量算力  
  
所以它能形成完整闭环：  
  
```  
模型  
↓  
Agent  
↓  
工作流  
↓  
企业应用  
```  
  
这就是 **ATH存在的意义**。  
  
   
  
## 五、但有一个关键假设  
  
所有分析都建立在一个前提之上：  
  
**大模型必须足够可靠。**  
  
也就是说：  
  
AI必须具备：  
  
* 逻辑能力  
* 稳定性  
* 长上下文  
* 低幻觉  
  
如果这个假设成立：  
  
AI工作流平台会爆发。  
  
   
  
## 六、如果这个假设崩塌呢？  
  
如果大模型始终存在：  
  
* 幻觉  
* 不稳定  
* 不可控  
  
那么 AI 就无法进入企业核心系统。  
  
那时候，AI的主要形态会变成：  
  
**辅助工具**  
  
例如：  
  
* Copilot  
* 助手  
* 自动化插件  
  
企业软件格局不会被颠覆。  
  
传统公司仍然会占主导。  
  
   
  
## 七、真正的大机会：AI + 工作系统  
  
但如果 AI 能稳定完成工作任务。  
  
那么未来企业软件形态会彻底改变。  
  
今天的软件结构：  
  
```  
菜单  
按钮  
表单  
```  
  
未来的软件结构：  
  
```  
对话  
任务  
Agent  
```  
  
人类只需要说一句话：  
  
> “帮我完成这个项目。”  
  
AI会自动：  
  
* 查数据  
* 写方案  
* 做分析  
* 安排流程  
  
这就是 **AI原生企业系统**。  
  
   
  
## 八、未来最大的 AI 公司是谁？  
  
很多人认为：  
  
未来 AI 霸主会是模型公司。  
  
但历史告诉我们：  
  
**真正的王者是平台公司。**  
  
移动互联网时代：  
  
* 操作系统：Apple、Google  
* 电商平台：Alibaba  
* 社交平台：Tencent  
  
AI时代也一样。  
  
未来真正的超级公司，大概率是：  
  
**AI工作平台。**  
  
谁掌控：  
  
* 企业数据  
* 工作流  
* AI Agent  
  
谁就掌控 **下一代生产系统**。  
  
   
  
## 结语  
  
很多人还在讨论：  
  
> AI是不是泡沫？  
  
但真正的问题其实是：  
  
> **AI什么时候进入生产系统。**  
  
阿里成立 **ATH + 悟空事业部**，其实已经给出了答案：  
  
**AI战争正在从“模型能力”转向“生产系统”。**  
  
一旦 AI 进入企业工作流。  
  
这将不是一次技术升级。  
  
而是一次：  
  
**工业革命级别的生产力重构。**
