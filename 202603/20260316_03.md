## 背景      
openrouter 的免费模型非常多, 而且都还不错.  
  
opencode 已内置一些免费模型, 而且不弱, 例如 kimi 2.5 pro, glm等.   
  
但是白嫖一直爽, 一直白嫖一直爽, 所以还可以通过 openrouter 的 free 模型来支持你的 opencode , 甚至 openClaw .  
  
假设你已经科学上网, 可以这样安装 opencode :   
```  
all_proxy=socks5h://127.0.0.1:11112 http_proxy=http://127.0.0.1:22222 https_proxy=http://127.0.0.1:22222 curl -fsSL https://opencode.ai/install | all_proxy=socks5h://127.0.0.1:11112 http_proxy=http://127.0.0.1:22222 https_proxy=http://127.0.0.1:22222 bash  
```

然后进入 openrouter 官网, 创建 API KEY.  
  
https://openrouter.ai/settings/keys  
  
把 key 记下来.  
  
创建工作目录, 进入工作目录, 启动 opencode, 配置 opencode API KEY :   
```  
mkdir ~/workspace_opencode  
cd ~/workspace_opencode  
git clone --depth 1 https://github.com/pgvector/pgvector  
all_proxy=socks5h://127.0.0.1:11112 http_proxy=http://127.0.0.1:22222 https_proxy=http://127.0.0.1:22222 opencode  
```  
  
配置 opencode 模型  
```  
ctrl+a 显示所有provider  
搜索 openrouter  
配置 openrouter ak  
配置模型: 所有免费模型都可以使用, 例如搜索 step free 模型, 选中 stepfun/step-3.5-flash:free   
```  
  
测试 opencode :  
```
# 下面的 prompt 建议还是在 pg docker 里搞, 因为已包含了 pg 数据库, 好调试.
```  
