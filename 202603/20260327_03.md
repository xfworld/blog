## 背景  
还记得前几天发的[《TradingAgents: Multi-Agents LLM Financial Trading Framework》](../202603/20260325_01.md)这篇论文的解读吗?  
  
这篇论文的核心，不是在发明某个新的技术指标，而是在把“一个交易团队怎么协作做决策”这件事，用多个 LLM 代理模拟出来。 它把“看基本面、看新闻、看情绪、看技术面、做多空辩论、控风险、最后下单”串成一个组织化流程。  
  
实际上这篇论文证明了一件事情: AI 多 Agent 组队分析决策股票交易, 这件事可行.  
  
而且"多 Agent 组队分析决策股票交易"项目开源了, 今天就来试一试这个开源项目.  
  
这个项目原来在这里 https://github.com/TauricResearch/TradingAgents  
  
但是不支持中文, 而且 openRouter 免费模型居然仅仅支持2个较老的模型, 所以我使用 codex 改了一下啊, 支持了所有的免费模型, 以及中文. 新版本已经放到了我自己的仓库中 https://github.com/digoal/TradingAgents  
  
现在分析更加靠谱了, 不得不说 codex 可真好用. 有需要 codex 的盆友欢迎私信联系, 刚好有盆友在卖官方账号.  
  
好, 我们继续, 本文将介绍如何使用 TradingAgents 做专业的股票分析.  
  
假设你已满足如下环境:  
```  
科学上网  
macOS  
python 3.13  
openRouter AK  
```  
  
---  
  
1、安装并启动 TradingAgents  
  
```  
git clone --depth 1 https://github.com/digoal/TradingAgents  
cd TradingAgents  
pip install .  
export OPENROUTER_API_KEY=...      # OpenRouter  
  
# 启动 TradingAgents  
all_proxy=socks5h://127.0.0.1:1111 http_proxy=http://127.0.0.1:22222 https_proxy=http://127.0.0.1:22222 python -m cli.main  
```  
  
2、分析股票代码  
  
- 输入股票代码: 例如 `GOOG`  
- 选择分析日期: 默认当日  
- 选择分析师团队: 例如全选 `market, social, news, fundamentals`  
- 选择分析深度: 有 3 档可选, 消耗的资源轮次可能不一样.  
- 选择模型供应商: 我这里选择的是 openRouter , 因为有很多模型可以白嫖  
- 选择快模型: [ Quick-Thinking LLM Engine ]: Step 3.5 Flash (free) - Fast reasoning with long context  
- 选择思考模型: [ Deep-Thinking LLM Engine ]: NVIDIA Nemotron 3 Super 120B A12B (free) - Strongest current free choice for deep work  
- 等待分析结果  
  
![pic](20260327_03_pic_001.jpg)  
  
分析完成, 可以保存完整报告到本地  
  
3、阅读报告  
  
完整报告非常详细, 包含了分析师团(市场分析师、社交舆情分析师、新闻分析师、基本面分析师)、研究团(看多研究、看空研究员、研究经理)、交易团队(交易员)的详细推理报告, 最终会给出终极决策.
