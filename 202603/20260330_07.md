```  
ANTHROPIC_AUTH_TOKEN  
ANTHROPIC_BASE_URL
```  

3、用 claude 给 TradingAgents 添加 MiniMax 模型供应商的支持  
  
```  
git clone --depth 1 https://github.com/digoal/TradingAgents  
cd TradingAgents  
claude  
```  
  
启动后，选择 信任此文件夹 (Trust This Folder)，以允许 Claude Code 访问该文件夹中的文件，随后开始在 Claude Code 中使用 MiniMax-M2.7  
```  
 Security guide  
  
 ❯ 1. Yes, I trust this folder  
   2. No, exit  
```  
  
提示词入下:  
````  
添加 minimax 模型供应商的支持, 参考环境变量入下  
```  
{  
  "env": {  
    "ANTHROPIC_BASE_URL": "https://api.minimaxi.com/anthropic",  
    "ANTHROPIC_AUTH_TOKEN": "MINIMAX_API_KEY",  
    "API_TIMEOUT_MS": "3000000",  
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": 1,  
    "ANTHROPIC_MODEL": "MiniMax-M2.7",

```
````  
